ETIQUETAS SEMÁNTICAS

Los elementos de una lista semantica son análogos a las divisiones, solo que su significado está implicito a la estructura. Se incluyen varias etiquetas de estructuración semantica tan importantes como: < header >, < footer >, < nav >, < section >, < article >, < aside >, < figure > ó < figcaption >, < details >, entre otras.

CSS GRID

El CSS Grid es un lenguaje que contiene funciones de diseño dirigidas a desarrolladores. Este se destaca porque permite que una página web sea dividida en áreas o regiones. Tal como en las tablas, el grid nos permite alinear los elementos en filas o columnas, pero también son posible muchos diseños más y más sencillo. Como los elementos secundarios se un contenedor se posicionan para que se solapen o se superpongan.
